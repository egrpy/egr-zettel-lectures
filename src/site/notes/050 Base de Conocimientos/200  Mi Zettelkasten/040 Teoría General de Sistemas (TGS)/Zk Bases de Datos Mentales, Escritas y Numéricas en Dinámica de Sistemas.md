## Bases de datos Mentales, Escritas y Numéricas en Dinámica de Sistemas

En [[050 Base de Conocimientos/200  Mi Zettelkasten/040 Teoría General de Sistemas (TGS)/Zk Dinámica de Sistemas – Definición y Alcance (Forrester)\|la visión de Forrester]], la construcción de modelos de [[050 Base de Conocimientos/200  Mi Zettelkasten/040 Teoría General de Sistemas (TGS)/Zk Dinámica de Sistemas\|Dinámica de Sistemas]] se fundamenta en la integración de tres grandes “bases de datos”: la base de datos mental, la base de datos escrita y la base de datos numérica [[050 Base de Conocimientos/900 Biblioteca/Zk Lit (Forrester, s.f.) Definition and Scope of System Dynamics\|(Forrester, s.f.)]]. Estas fuentes no son alternativas excluyentes, sino componentes complementarios de un mismo proceso de modelado.

### Bases de Datos Mentales
La **base de datos mental** recoge el conocimiento tácito de los actores del sistema: experiencia acumulada, reglas no escritas, cultura organizacional y juicios prácticos que guían las decisiones cotidianas [[050 Base de Conocimientos/900 Biblioteca/Zk Lit (Forrester, s.f.) Definition and Scope of System Dynamics\|(Forrester, s.f.)]]-

### Base de Datos Escrita
La **base de datos escrita** incluye documentos formales como informes, normativa, procedimientos, contratos y estudios técnicos, donde se explicitan políticas y estructuras de decisión [[050 Base de Conocimientos/900 Biblioteca/Zk Lit (Forrester, s.f.) Definition and Scope of System Dynamics\|(Forrester, s.f.)]].

### Base de Datos Numéricas
La **base de datos numérica** corresponde a las mediciones y registros cuantitativos (series temporales, indicadores, estadísticas) que permiten calibrar y contrastar el comportamiento del modelo con la realidad observada [[050 Base de Conocimientos/900 Biblioteca/Zk Lit (Forrester, s.f.) Definition and Scope of System Dynamics\|(Forrester, s.f.)]].

Un modelo de Dinámica de Sistemas es más robusto cuando articula de forma coherente estas tres bases de datos, en lugar de apoyarse únicamente en datos numéricos. La base mental aporta comprensión contextual y sentido a las relaciones causales, la base escrita ofrece trazabilidad de las reglas formales, y la base numérica permite validar patrones de comportamiento en el tiempo. Desde la perspectiva de la [[050 Base de Conocimientos/200  Mi Zettelkasten/100 Docencia/Org1/2025/Clase 02 Introducción a la Teoría General de Sistemas/Zk Enfoque de la Teoría General de Sistemas\|Teoría General de Sistemas]], esta integración reduce la [[050 Base de Conocimientos/200  Mi Zettelkasten/100 Docencia/Org1/2025/Clase 06 Sistemas, Fundamentos, Propiedades, Principios Básicos/Zk Entropía\|entropía]] informacional del modelo y mejora su capacidad para representar la complejidad de los [[050 Base de Conocimientos/200  Mi Zettelkasten/100 Docencia/Org1/2025/Clase 06 Sistemas, Fundamentos, Propiedades, Principios Básicos/Zk Sistemas (Complejidad de los Sistemas)\|sistemas abiertos]].
