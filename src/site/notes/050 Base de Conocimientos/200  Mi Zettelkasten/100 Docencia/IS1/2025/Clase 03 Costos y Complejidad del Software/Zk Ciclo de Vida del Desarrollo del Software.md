## Ciclo de Vida del Desarrollo del Software
El Ciclo Ciclo de Vida del Desarrollo del Software (SDLC) es un proceso fundamental en la ingeniería de software, abarca las diversas fases, desde la concepción inicial hasta el mantenimiento del producto final. Las fases típicas incluyen [[050 Base de Conocimientos/900 Biblioteca/Zk Lit (Pressman, 2013) Ingeniería del Software - Un Enfoque Práctico (Séptima edición). McGraw-Hill Education\|(Pressman, 2013)]], [[050 Base de Conocimientos/900 Biblioteca/Zk Lit (Weitzenfeld, 2005) Ingenieria de software orientada a objetos con UML, Java e Internet\|Weitzenfeld, 2005)]]:

1. Identificación del Problema
2. Planificación del Proyecto
3. Captura de Requerimientos
4. Análisis de Requerimientos
5. Diseño del sistema
6. Implementación o Desarrollo
7. Pruebas
8. Despliegue o Implementación
9. Mantenimiento
